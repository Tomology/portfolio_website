Background

Cimex was created to be a project management application. The goal was to create an application that would allow users to track and manage their projects, as well as to connect and collaborate with others. I had been learning the MERN stack and was itching to put into practice some of the concepts I had learned. Building a bug tracker allowed me not only to create a useful app but helped me familiarize myself with a tool I would be using as a developer.

Features

Cimex allows registered users to create, update and share projects, track and manage issues on those projects, as well as to collaborate with others by creating teams. Issues can be assigned to any user with whom the project is shared while custom middleware will prevent unauthorized users from accessing your projects. A simple e-mail search allows other users to be added to your contacts.
